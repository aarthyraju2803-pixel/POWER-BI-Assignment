Project Overview

This project was developed using Microsoft Power BI to analyze an E-Commerce Sales Dataset. The objective of the assignment was to perform data preparation, data modeling, DAX calculations, and interactive data visualization to generate meaningful business insights related to sales performance, profitability, targets, and regional analysis.

The project demonstrates the practical implementation of business intelligence concepts using Power BI and DAX.

Dataset Used

The project includes the following datasets:

List of Orders
Order Details
Sales Target

These datasets contain information related to orders, sales amount, profit, product categories, quantity, customer location, and sales targets.
